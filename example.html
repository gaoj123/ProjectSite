Iteration 0

Each page starts out with 1/n as its PR, where n is the number of pages


Iteration 1

Iteration 1
A1/12
B2.5/12
C4.5/12
D4/12

PR(A) = PR(C)/C(C) = (1/4)/3 = 1/12

          PR of C is 1/4 ; C has 3 outgoing links

PR(B) = PR(A)/C(A) + PR(C)/C(C) = (1/4)/2 +(1/4)/3 = 2.5/12

PR(C) = PR(A)/C(A) + PR(D)/C(D) = (1/4)/2 + (1/4)/1 = 4.5/12

PR(D) = PR(C)/C(C) + PR(B)/C(B) = (1/4)/3 + (1/4)/1 = 4/12


Iteration 2

Iteration 2
A1.5/12
B2/12
C4.5/12
D4/12

PR(A) = PR(C)/C(C) = (4.5/12)/3 = 1.5/12

         PR of C is 4.5/12; C has 3 outgoing links

PR(B) = PR(A)/C(A) + PR(C)/C(C) = (1/12)/2 +(4.5/12)/3 = 2/12

PR(C) = PR(A)/C(A) + PR(D)/C(D) = (1/12)/2 + (4/12)/1 = 4.5/12

PR(D) = PR(C)/C(C) + PR(B)/C(B) = (4.5/12)/3 + (2.5/12)/1 = 4/12


We can keep repeating this procedure to get more stable values but let's look at ranks so far (Iteration 2).

Higher values mean greater importance of webpages.

Therefore, the pages in order of importance are: C, D, B, and A.
